# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **security@appstrate.dev** with a description of the
vulnerability, steps to reproduce, and any relevant details.

We will acknowledge receipt within 48 hours and provide a detailed
response within 7 days, including next steps and timeline for a fix.

## Safe Harbor

We consider security research conducted in good faith to be authorized.
We will not pursue legal action against researchers who follow this policy.
